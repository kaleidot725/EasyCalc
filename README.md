# 概要

Androidで暗算を簡単に楽しくやるためのアプリを開発するためのリポジトリです。

# 主な機能

- たし算・ひき算・わり算・かけ算・九九の計算問題を出題する
- 出題した計算問題の結果を表示して答え合わせをする
